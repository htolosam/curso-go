## Requisitos

- Go 1.20 o superior instalado
- Editor de código (VS Code, GoLand, etc.)

## Cómo usar este curso

1. Clona este repositorio
2. Navega a cada carpeta de ejemplo
3. Lee los comentarios en el código
4. Ejecuta los ejemplos con `go run`
5. Practica modificando el código
