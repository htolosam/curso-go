# Control de Flujo

Estructuras de control en Go: if, switch, for, y defer.

## Ejecutar

```bash
go run main.go
```

## Conceptos

- `if/else`: Condicionales simples y anidados
- `switch`: Casos múltiples y sin condición
- `for`: Bucles clásicos, tipo while, infinitos
- `range`: Iteración sobre colecciones
- `break` y `continue`
- `defer`: Ejecución diferida
