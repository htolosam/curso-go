# Structs

Los structs son tipos de datos compuestos que agrupan campos relacionados.

## Ejecutar

```bash
go run main.go
```

## Conceptos

- Definir y crear structs
- Acceso y modificación de campos
- Punteros a structs
- Structs anidados
- Composición (embedding)
- Métodos de structs
- Comparación de structs
- Tags de structs
