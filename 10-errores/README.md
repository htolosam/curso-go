# Manejo de Errores

Go maneja errores como valores explícitos, no como excepciones.

## Ejecutar

```bash
go run main.go
```

## Conceptos

- Crear errores con `errors.New` y `fmt.Errorf`
- Verificar errores
- Errores personalizados
- Múltiples errores
- Wrapping errors (Go 1.13+)
- Sentinel errors
- Panic y recover
- Mejores prácticas
