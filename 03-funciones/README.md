# Funciones

Aprende sobre funciones en Go: declaración, parámetros, retornos y características avanzadas.

## Ejecutar

```bash
go run main.go
```

## Conceptos

- Funciones básicas
- Parámetros y retornos
- Múltiples valores de retorno
- Retornos nombrados
- Funciones variádicas
- Closures (funciones que retornan funciones)
- Recursión
