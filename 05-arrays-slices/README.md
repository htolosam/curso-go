# Arrays y Slices

Aprende sobre arrays (tamaño fijo) y slices (tamaño dinámico) en Go.

## Ejecutar

```bash
go run main.go
```

## Conceptos

- Arrays: colecciones de tamaño fijo
- Slices: colecciones dinámicas
- Operaciones: append, copy, slicing
- Longitud y capacidad
- Slices multidimensionales
- Diferencia entre nil y vacío
