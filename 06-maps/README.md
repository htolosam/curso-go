# Maps

Los maps son colecciones de pares clave-valor en Go (similares a diccionarios o hash tables).

## Ejecutar

```bash
go run main.go
```

## Conceptos

- Crear maps con make y literales
- Agregar, modificar y eliminar elementos
- Verificar existencia de claves
- Iterar sobre maps
- Maps anidados
- Maps con structs y slices
- Map nil
